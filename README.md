# tg-radio-bot

بوت راديو بقائمة عامة + حساب مساعد (User) للبث داخل Voice Chat/Live.

## Variables
- BOT_TOKEN
- API_ID
- API_HASH
- ASSISTANT_SESSION
- DATABASE_URL
- DEVS=123,456
- DEFAULT_LOOP=1
- AUTO_START=0

## How to add songs
Send audio/voice to the bot in private (devs only) -> saved automatically.

## Run
python -m bot.main
